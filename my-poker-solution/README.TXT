========================
POKER HAND SORTER
========================
1. Use Netbeans to open the downloaded project to view the source code. As the project was implemented using java in the Netbeans IDE.
2. The latest solution "my-poker-solution.jar" file is already present in the "dist" folder inside the "my-poker-solution" folder.
3. Open Windows Powershell.
4. Use the "cd" command to set the path to "dist" folder which is in the "my-poker-solution" folder.
5. Type the command "java -jar my-poker-soultion.jar".
6. The terminal will show the desired output along with a seperate "output.txt" created in the "dist" folder consisting the solution.
